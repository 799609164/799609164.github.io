SpringBoot 框架

官方文档

1 简介

  • SpringBoot 是一个基于 Spring 开发的 javaweb 的框架，以约定大于配置的核心思想，默认进行了很多设置
  • 优点
    • 使用 Spring Initializr 可以快速配置一个 Spring Boot 应用
    • 对大量的框架都可以无缝集成，很少的配置就可以运行
    • 简化的安全性
    • 丰富的 SQL 和 NoSQL 支持
    • 嵌入式 Servlet 容器的支持；如：Tomcat，Jetty 和 Undertow。
    • 提高开发、生产效率，如热部署等功能
    • 提供监控功能，如跟踪，指标和健康状况
